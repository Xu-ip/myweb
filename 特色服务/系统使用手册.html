快速入门

👤

注册登录

首次使用需要注册账号，填写基本信息后即可完成注册。注册后使用手机号或邮箱登录系统。

1. 点击首页"注册"按钮 2. 填写注册信息 3. 验证邮箱/手机 4. 登录系统
🖥️

界面介绍

系统主界面包括导航栏、搜索区、内容区等。导航栏用于页面切换，搜索区用于快速查找内容，内容区显示各类信息。

导航栏：页面切换 搜索区：快速查找 内容区：信息展示 工具栏：快捷操作
⌨️

基本操作

系统支持鼠标点击操作，大部分功能通过点击即可完成。部分功能支持键盘快捷键，提高操作效率。

Ctrl + F：搜索 Ctrl + S：保存 Alt + ←：返回 Alt + →：前进

常见问题

如何修改个人信息？

+

登录后进入个人中心，点击"个人信息"即可修改基本信息。修改完成后点击"保存"按钮即可生效。

如何下载数据？

+

在数据详情页面点击"下载"按钮，选择保存位置即可下载数据文件。支持多种格式导出，包括Excel、CSV等。

如何订阅服务？

+

进入订阅服务页面，选择合适的套餐，填写信息并完成支付即可订阅。订阅后可享受相应的服务权益。

忘记密码怎么办？

+

在登录页面点击"忘记密码"，通过手机或邮箱验证后可重置密码。建议设置强密码并定期更新。

如何联系技术支持？

+

可以通过在线客服、电话支持、邮件支持或工单系统联系技术支持。工作时间内将优先处理您的问题。
